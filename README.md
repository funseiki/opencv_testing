Open CV Testing
===============

Setting up
----------
	1) Copy opencv/build/include to opencv/build/x86/vc11/
	2) Create an environment variable named "OPENCV_32"
	3) Set the value to be "path/to/opencv/build/x86/vc11/"
	4) Add %OPENCV_32%/bin/ to your PATH
